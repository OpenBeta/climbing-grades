# What is this?

Javascript utilities for working with rock climbing grades.

*Only YDS and V scale are supported at the moment. Code contributions are most welcome!*

Questions?  Join us on [Discord](https://discord.gg/fY9DbRav8h).

### Development

```
yarn install
yarn test
```
